# Ogre Tools

An assortment of tools for brutal software development.

## Collaboration

```
$ npm install

$ npm test

$ npm run test:ci

$ npm run build-all-packages
```

## Packages

| Package | Description |
|---------|-------------|
| [`@lensapp/injectable`](https://www.npmjs.com/package/@lensapp/injectable) | A brutal dependency injection container |
| [`@lensapp/injectable-react`](https://www.npmjs.com/package/@lensapp/injectable-react) | A brutal component for injecting components that use injectable |
| [`@lensapp/injectable-extension-for-auto-registration`](https://www.npmjs.com/package/@lensapp/injectable-extension-for-auto-registration) | Auto-registration of injectables for Injectable in Ogre Tools |
| [`@lensapp/fp`](https://www.npmjs.com/package/@lensapp/fp) | A collection of low abstraction functions in functional programming flavor |
| [`@lensapp/generable`](https://www.npmjs.com/package/@lensapp/generable) | A collection of low abstraction functions in functional programming flavor for generators |
| [`@lensapp/mobx-utils`](https://www.npmjs.com/package/@lensapp/mobx-utils) | MobX utilities for Ogre Tools |
| [`@lensapp/element-component`](https://www.npmjs.com/package/@lensapp/element-component) | Re-usable props in React |
| [`@lensapp/discoverable`](https://www.npmjs.com/package/@lensapp/discoverable) | A test-utility to control HTML-elements using data-attributes |
| [`@lensapp/ogre-test-utils`](https://www.npmjs.com/package/@lensapp/ogre-test-utils) | A collection of test-utils for Ogre Tools |
| [`@lensapp/linkable`](https://www.npmjs.com/package/@lensapp/linkable) | Like 'npm link', but doesn't trip over with symlinks and local node_modules of linking target |
